# Changelog

## [0.1.0] — 2026-03-09

### Added

- Initial app scaffold with Lovable
- Authentication: email + Google OAuth via Lovable Cloud
- Onboarding flow: Welcome, About You, Goals, Diet & Nutrition screens
- TDEE calculation using Mifflin-St Jeor equation
- Imperial/Metric unit toggle with live conversion
- BMI gradient bar with color-coded health labels
- 4 goal cards with dynamic calorie and macro updates
- Cuisine style multiselect with custom tag input
- Complexity level selector (Super Simple / Home Chef / Master Chef)
- Meal weight preference (Morning / Midday / Evening)
- Healthy swaps toggle
- Protein, carb, veggie multiselect pill tags
- Fat single-select pills
- AI meal plan generation via Gemini (google/gemini-2.5-flash with fallback)
- Chain-of-thought calorie calculation from USDA values
- 7-day meal plan display with day-cards and meal tiles
- Macro badges and day total summary rows
- Lock icon for meal preservation on regeneration
- Heart icon to save meals to Vault
- Meal Vault tab with staples system (max 3)
- Grocery List screen with category grouping
- Cook Guide screen with parallel cooking steps
- Sunday batch cook choreography
- Sticky footer (Grocery List + Cook Guide)
- Bottom tab bar (This Week + Vault)
- Right slide-in side sheet with profile navigation
- 4 profile edit screens (My Profile, About You, My Goals, Diet & Nutrition)
- Regeneration bottom sheet after goal changes
- Dark mode toggle with warm color tokens
- Price and timing transparency tooltips
- GitHub integration for version control
